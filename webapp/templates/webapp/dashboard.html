Auto-flag criteria

An item is flagged for deletion when all of the following are true:

  • File size > 0 — item has files on disk (not just in Radarr/Sonarr)
  • Never watched — no play history in Tautulli
  • IMDB rating < {{ IMDB_RATING_THRESHOLD }} — below the configured threshold
  • Popularity ≤ 10 — not trending (movies only)
  • Added more than {{ RECENTLY_ADDED_MONTHS }} months ago — not recently added
  • Released more than {{ RECENTLY_ADDED_MONTHS }} months ago — not a recent release

Items with unknown ratings (N/A) are never auto-flagged. If any criterion is green in the table, that factor alone prevents auto-flagging.
